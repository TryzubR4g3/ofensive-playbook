# nuclei

Template-based scanner for quick checks against known exposures, CVEs, and common web misconfigurations.

## Commands Used

### Scan a web target

```bash
nuclei -target http://internal.thm/
```

Used on: **Internal** - identified WordPress, XML-RPC exposure, a user enumeration finding, and an SSH Terrapin finding.
